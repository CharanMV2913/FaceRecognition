        <h2>Overview</h2>

        <p>This project is a face recognition system designed to generate samples, train a model, and recognize faces.</p>

        <h2>üöÄ Getting Started</h2>

        <h3>Sample Generation</h3>

        <ol>
            <li>Open <code>samplegenerator.py</code>.</li>
            <li>Enter a unique ID that is not used.</li>
            <li>Enter the name of the person.</li>
            <li>It will take some time to create samples for you.</li>
        </ol>

        <h3>Model Training</h3>

        <ol>
            <li>Open <code>modeltrainer.py</code>.</li>
            <li>Run the script.</li>
            <li>Wait for the message "Model trained, Now we can recognize your face."</li>
        </ol>

        <p>Your model is now trained and ready for recognition.</p>

        <h3>Face Recognition</h3>

        <p>Run <code>facerecognition.py</code>.</p>
